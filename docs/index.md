# Phylogenetic Tree Building Tutorial

This project demonstrates building phylogenetic trees using the UPGMA algorithm with flexible distance metrics.

## Getting Started

### Installation

```bash
pip install -r requirements.txt
```

### Local Documentation

To view this documentation locally:

```bash
pip install mkdocs-material
mkdocs serve
```

Then navigate to http://127.0.0.1:8000
